# Furnace

## Purpose
Furnace module used in the AC-OTFlex system.

The firmware in this folder controls the linear actuator used to open the furnace door.

## Structure
- `firmware/` - Device firmware
- `mechanical/` - CAD models and mechanical documentation
- `electrical/` - Wiring and electrical schematics
- `documentation/` - Usage and specifications

## Status
Active hardware module. See firmware and wiring docs for setup and troubleshooting.
