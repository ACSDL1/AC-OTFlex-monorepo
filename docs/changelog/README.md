# Project Changelog

This directory contains detailed changelogs organized by contributor. Each contributor has their own folder documenting the major changes, features, and improvements they contributed to the project.

## Contributor Folders

- **Iliya/** - Original system architecture, Arduino integration, and JSON workflow system
- **Alan-Yang/** - OT2 support, MQTT IoT infrastructure, and power management
- **Gavin-Tranquilino/** - Monorepo structure, documentation, and Jupyter notebook workflows

## How to Document Your Contributions

When making changes to the project:

1. Create entries in your personal changelog folder
2. Use clear, descriptive filenames for different features:
   - `MQTT_implementation.md`
   - `Device_integration_XYZ.md`
   - `Documentation_updates.md`
3. Keep entries dated and include feature branches/commit references
4. Update your folder's `README.md` periodically with major milestones
